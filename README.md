#  Global Fashion Retail Analytics (SQL + Insights)

## 📌 Project Overview
This project simulates an end-to-end data analytics workflow for a multinational fashion retail company. Using SQL, the analysis transforms raw transactional and customer data into actionable business insights.

The project focuses on sales performance, customer segmentation, promotion effectiveness, and operational efficiency.

---

## 🎯 Business Objectives
The analysis aims to:
- Optimize product assortment and pricing strategies
- Identify high-value customers and improve retention
- Measure promotional effectiveness
- Reduce returns and operational inefficiencies

---

## 🗂️ Dataset Description
The dataset includes multiple interconnected tables:

- **Transactions**: Sales, returns, payment methods, store details
- **Customers**: Demographics and geographic data
- **Products**: Categories, brands, pricing
- **Discounts**: Promotion periods and discount details

## 🔍 Key Analyses Performed

### 1. Customer Demographics Analysis
- Unique customers by country and city
- Average purchase value per segment

### 2. Sales vs Returns Trends
- Monthly return percentage
- Return behavior tracking over time

### 3. Category Performance
- Top categories by revenue
- Contribution to total sales

### 4. Payment Method Insights
- Sales volume by payment type
- Average basket value comparison

### 5. Promotion Impact Analysis
- Revenue comparison: discount vs non-discount periods
- Effectiveness of promotional campaigns

### 6. Customer Loyalty Tracking
- New vs repeat customers (monthly)
- Average spend comparison

### 7. Store Performance Ranking
- Top and bottom stores by revenue
- Return rate analysis per store

### 8. Price Band Analysis
- Product segmentation into price ranges
- Revenue and return contribution per band

### 9. Basket Size Insights
- Average items per transaction
- Revenue per basket across countries

### 10. Growth Trends
- Month-over-month revenue growth
- Growth analysis for top 3 categories

## 🚀 Key SQL Concepts Used
- Complex Joins across multiple tables
- Window Functions (ROW_NUMBER, RANK, LAG)
- Aggregations & Grouping
- Case-based segmentation (Price bands, customer types)
- Time-based analysis (monthly trends)
- Percentage and ratio calculations
